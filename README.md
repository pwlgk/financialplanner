# Financial Planner 💰

Приложение для учета личных финансов "Financial Planner" (Дипломный проект).
Позволяет удобно отслеживать доходы, расходы, планировать бюджет и достигать финансовых целей.

## Основные возможности 🚀

- **Дашборд:** Общий обзор финансового состояния, баланс и последние транзакции.
- **Доходы и Расходы:** Учет финансовых операций (incomes / expenses).
- **Категории:** Гибкое управление категориями доходов и расходов.
- **Бюджет:** Планирование и контроль ежемесячного бюджета.
- **Финансовые цели:** Установка целей и отслеживание прогресса их достижения.
- **Отчеты:** Визуализация данных в виде графиков и диаграмм для анализа финансового поведения.

## Стек технологий 🛠️

Приложение разработано с использованием современного стека технологий Android:

- **Язык программирования:** Kotlin
- **Android SDK:** `minSdk = 24` (Android 7.0), `targetSdk = 34` (Android 14)
- **Архитектура:** MVVM (Model-View-ViewModel) + Repository Pattern
- **UI & Навигация:**
  - ViewBinding для безопасного доступа к элементам UI.
  - Navigation Component (с поддержкой Safe Args) для управления переходами.
  - Material Design Components для построения современного пользовательского интерфейса.
- **Многопоточность:** Kotlin Coroutines & Flow
- **База данных:** Room Database (с KSP)
- **Графики:** MPAndroidChart для построения диаграмм в отчетах.

## Архитектура проекта 🏗️

Проект структурирован по пакетам в соответствии с функциональными слоями:
- `data/`: Слой работы с данными (Database, Models, Repositories).
- `ui/`: Слой пользовательского интерфейса (Activities, Fragments, ViewModels, Adapters). Разбит на подпакеты по экранам (`dashboard/`, `expense/`, `income/`, `budget/`, `goals/`, `reports/`, `categories/`).
- `util/`: Утилитарные классы и функции-расширения.

## Установка и запуск 💻

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/pwlgk/financialplanner.git
   ```
2. Откройте проект в **Android Studio** (рекомендуется последняя стабильная версия).
3. Дождитесь завершения Gradle Sync.
4. Выберите эмулятор или подключенное Android устройство (API >= 24) и нажмите **Run** (Shift + F10).
